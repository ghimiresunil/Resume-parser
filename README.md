# Resume-parser
A prototype resume parser for parsing the CVs

## Features

- Extract Name
- Extract Email
- Extract Mobile Number
- Extract Skills
- Extract Soft skills
- Extract Nationality
- Extract Language
- Extract Total Experience
- Extract College Name
- Extract Degree
- Extract Designation
- Extract Company Names
- Github Link
- Linkedin Link
- Date of Birth
- Gender
- Zipcode
- Project
- Rewards
- References
